<h2>介绍</h2>
1.是一个基于WMP多媒体控件开发的音乐播放器。
<h2>功能</h2>
1.支持mp3,mp4,wav等各种格式。<br/>
2.支持歌词显示。<br/>
3.支持文件夹添加歌曲。<br/>
<h2>运行</h2>
1.下载运行里面的exe文件。
